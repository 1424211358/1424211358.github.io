地址：https://adworld.xctf.org.cn/adw/selected_topic/?type=misc

下载了一个名为linux的文件，扔进kaili

1.查看字符串 ：root@kali2017:~/桌面# strings linux | grep flag

2.使用mount将文件挂载

3.将文件内容输出 root@kali2017:~/桌面# cat /mnt/O7avZhikgKgbF/flag.txt

​ ZmxhZ3tzYWpiY2lienNrampjbmJoc2J2Y2pianN6Y3N6Ymt6an0=

4.发现是一串base64编码，解码 root@kali2017:~/桌面# base64 -d /mnt/O7avZhikgKgbF/flag.txt
