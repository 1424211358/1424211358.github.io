安装黑苹果

需准备8G内存以上的u盘，安装DISKGenius，etcher，EasyUEFI

echer用来写入mac镜像

DIskGenius用来隔离磁盘，和替换clover

EasyUEFI 替换引导

1.打开etcher select image 选择镜像 -> select drive 等待完成

2.隔离100G空间给mac，在此分区中新建一个efi分区

3..进入BIOS 选择U盘启动，安装mac

4.抹除磁盘

免杀
